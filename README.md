## 📖 프로젝트 소개 (About Project)

**"2026 Lucky Universe"**는 최신 생성형 AI 기술을 활용하여  
사용자 맞춤형 운세와 행운 부적(럭키 다마고치)을 제공하는 웹 애플리케이션입니다.

단순한 랜덤 운세가 아닌,  
Google Gemini 2.5 Flash 모델이 사용자의 **생년월일과 MBTI를 분석**하여  
세상에 단 하나뿐인 조언과 캐릭터를 생성합니다.

---

## 🎯 기획 의도

- **초개인화 경험:** MBTI + 사주 기반 운세 제공  
- **AI 인터랙션:** Pixel Art ‘럭키 다마고치’ 생성  
- **데일리 재미 요소:** 행운 점수 / 미션 / 로또 번호  
- **모바일 퍼스트 UI 디자인**

---

## ✨ 주요 기능 (Features)

### 🔮 AI 종합 운세 분석  
2026년 총운, 재물운, 연애운, 직업운, 건강운을 세부 분석

### 👾 럭키 다마고치 생성  
사용자의 소원과 MBTI를 기반으로 한 Pixel Art / SVG 캐릭터 생성

### 🎰 데일리 게이미피케이션  
- 행운 점수  
- 데일리 행운 미션  
- AI 로또 번호 추첨  

### 📱 반응형 UI/UX  
모바일·PC에서 모두 최적화
