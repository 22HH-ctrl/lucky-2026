📖 프로젝트 소개 (About Project)

2026 Lucky Universe는 최신 Google Gemini AI 기술을 활용해
사용자에게 초개인화된 2026년 운세 분석과
나만의 ‘럭키 다마고치’ 캐릭터를 생성해주는 웹 애플리케이션입니다.

🎁 사용자 생년월일 + MBTI → AI가 분석해서 ‘세상에 하나뿐인 운세/캐릭터 생성’

🎯 기획 의도 (Project Goals)

초개인화 경험 제공
사용자 MBTI + 사주 기반으로 개성 있는 MZ 스타일 운세 제공

AI 인터랙션 강화
텍스트 운세뿐만 아니라 소원을 반영한 픽셀 아트 다마고치 캐릭터 생성

엔터테인먼트 요소 추가
매일 바뀌는 행운 점수, 미션, 로또 번호 등 재미 요소 강화

🚀 실행 방법 (Getting Started)
1. 저장소 클론
git clone https://github.com/your-username/lucky-2026.git
cd lucky-2026

2. 패키지 설치
npm install

3. 환경 변수 설정

루트 폴더에 .env 파일 생성 후 아래 입력:

VITE_API_KEY=your_gemini_api_key_here

4. 로컬 실행
npm run dev
